Como Usar

🎤
Chat por Voz
Clique no microfone ou pressione Espaço. Sky responde com voz. Escape para parar.
💬
Chat por Texto
Use "Chat IA Texto" na barra lateral ou o campo de texto na tela de voz. Enter para enviar.
📷
Câmera
Botão "Ver Câmera" na barra do chat de voz. Capture e Sky analisa a imagem.
📎
Arquivos
Envie imagens ou .txt para análise pelo botão "Analisar Arquivo".
🧠
Memória
Sky aprende sobre você automaticamente. Veja em "Memória" (ícone banco de dados).
⚡
Conversa Contínua
Ative para Sky ouvir e responder automaticamente sem precisar clicar.
🔑
Chaves API
Configure em Configurações e Notificações. Gemini obrigatório, ElevenLabs opcional.

Suporte e Instalação

⚙️
Requisitos
Chrome ou Edge para reconhecimento de voz. Node.js 18+ para servidor local.
🔑
Chave Gemini API
Obtenha grátis em aistudio.google.com — insira em Configurações e Notificações.
🎙️
ElevenLabs (opcional)
elevenlabs.io → copie a API key → Configurações. Ativa a voz Jarvis premium em pt-BR.
🚀
Iniciar localmente
No terminal da pasta do projeto: npx serve . — acesse localhost:3000
🐛
Microfone não funciona
Use Chrome/Edge. Clique no cadeado na barra de endereço e permita o microfone.
